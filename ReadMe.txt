This music player uses the following libraries that needs to be imported before running:

Default Libraries
	-npm install

Expo Media Library
	-npx expo install expo-media-library
React Navigation
	-npm install @react-navigation/native
	-npx expo install react-native-screens react-native-safe-area-context
	-npm install @react-navigation/bottom-tabs
Icons Used from
	-icons.expo.fyi

RecylerView Components:{This Component is created by Author naqvithla and is used from his public git repo}
This Component Allows us to have a performance based output of all the audio files list to optimize the app rather than showcasing individual item that can take a long time.
	-npm install --save recyclerlistview
